# SC-TH 主线说明

当前产品已经从 LaTeX review workspace 重构为极简 Word 导出工具。

## 当前主路径

1. 首页只保留 `SC-TH`、一个输入框、`+` 上传入口、错误提示区和清空入口。
2. 用户上传 `.docx` 或粘贴文本后，触发初检。
3. 系统返回预检确认弹窗，展示结构预览与问题分层。
4. 只有阻塞项清零后，才允许确认导出。
5. 导出阶段用浅蓝海浪进度反馈，完成后自动下载 `.docx`。

## 规则层

- 阻塞项：不满足时禁止导出
- 警告项：允许继续，但提示风险
- 信息项：解释当前识别结果和导出预期

当前阻塞重点是题目、中文摘要、正文结构、正文长度、参考文献和模板可用性。

## 当前边界

- 仅支持本科论文入口
- 仅公开 `.docx` 主产物
- 不保留旧 review workspace 作为主路径
- 不承诺图片、表格、脚注和复杂排版高保真迁移
- 不承诺引用链 100% 自动对齐
