# SC-TH 本地验收清单

## 主链路

1. 打开首页
2. 检查上传 `.docx` 与粘贴文本入口都可用
3. 触发预检，确认弹窗展示“缺失章节留白”和“复杂元素需人工复核”
4. 通过预检后导出 `.docx`
5. 运行 `uv run python scripts/check_docx_compliance.py <导出文件>`
6. 用 Word 打开并更新目录

## 重点抽查

- 正式封面是否为第一页
- 封面字段缺失是否保留空白槽位
- 中文摘要 / 英文摘要 / 目录 / 正文 / 参考文献 / 附录 / 致谢顺序是否正确
- 目录是否为 Word 字段
- 页眉是否使用主标题且不换行
- 前置部分与正文的页码体系是否正确
- 参考文献是否有统一悬挂缩进

## 异常链路

1. 空输入提交
2. 非 `.docx` 上传
3. 缺少摘要但正文存在
4. 缺少参考文献 / 附录 / 致谢
5. 包含表格 / 图片 / 脚注的 `.docx`
