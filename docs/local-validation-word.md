# SC-TH 本地验收清单

## 主链路

1. 打开首页
2. 检查页面只显示 `SC-TH` 与一条主输入框，不再显示旧的大标题
3. 检查默认提示为 `按 Cmd/Ctrl + Enter 开始预检`
4. 检查空态输入框高度接近单条输入栏，长文本不会迅速把首页撑高
5. 上传合法 `.docx`
6. 弹出预检确认弹窗
7. 点击取消，确认输入仍保留
8. 再次触发预检并确认
9. 进入海浪导出进度
10. 自动下载 `.docx`

## 文本链路

1. 粘贴完整论文文本
2. `Cmd/Ctrl + Enter` 触发预检
3. 检查输入框只在前几行内渐进扩展，超过阈值后改为内部滚动
4. 检查题目、摘要、正文结构、参考文献的块级预览
5. 通过预检后成功导出

## 异常链路

1. 空输入提交
2. 非 `.docx` 上传
3. 中文摘要不足
4. 正文长度不足
5. 参考文献缺失
6. 模板缺失
