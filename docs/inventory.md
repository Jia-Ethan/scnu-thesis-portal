# 模板盘点

本文档用于记录当前仓库中三套模板的基础情况、角色判断与后续优先级。

## 1. `scnuthesis-scnu`

- 角色标签：`primary-candidate`
- 面向人群：硕士 / 博士为主，README 同时提到存在本科分支
- 上游来源：`https://github.com/scnu/scnuthesis`
- 最近提交：`2024-12-30`
- 编译入口：`thesis.tex`

### 明显优点

- 上游近期仍有更新，活跃度相对更好。
- README 明示采用 LPPL，授权信息相对清晰。
- 模板说明、选项和示例相对完整，适合作为后续主线参考。

### 明显限制

- 当前收集到的是硕博主模板，不等于直接覆盖本科场景。
- README 仍强调“非官方模板”，不能视作学校官方结论。
- 暂未完成本地编译验证和官方格式逐条比对。

## 2. `latex-scnu`

- 角色标签：`reference`
- 面向人群：本科毕业论文用户
- 上游来源：`https://github.com/claviering/latex-scnu`
- 最近提交：`2019-04-20`
- 编译入口：`main.tex`

### 明显优点

- 目录结构直观，适合作为本科场景的轻量参考。
- 模板组织接近“摘要、正文、附录、参考文献、致谢”的常见写作流程。
- 学习成本相对较低。

### 明显限制

- 上游明显较旧，持续维护迹象较弱。
- 授权信息不够清晰，不适合在未核实前当作统一主线。
- 更像一个单一模板仓库，不足以单独支撑 Portal 的长期产品方向。

## 3. `scnuthesis-zsben2`

- 角色标签：`reference`
- 面向人群：研究生用户，偏熟悉 LaTeX 的进阶使用者
- 上游来源：`https://github.com/zsben2/scnuthesis`
- 最近提交：`2023-06-10`
- 编译入口：`main.tex`

### 明显优点

- 技术栈较新，基于 LaTeX3 方向实现。
- 结构中体现了电子版、盲审、打印等场景意识。
- 可作为研究生模板设计和演进方向的对照样本。

### 明显限制

- 上游活跃度和生态影响力不如 `scnuthesis-scnu`。
- 当前信息显示其更偏研究生场景，不适合作为全局默认入口。
- 授权信息同样需要后续专门核实。

## 当前判断

- 现阶段最合理的仓库定位，不是单纯“模板集合”，也不是已经成型的“一站式论文网站”。
- 它更适合作为“论文模板整合与使用入口仓库 / Portal 雏形”。
- `scnuthesis-scnu` 适合做后续主线参考模板。
- `latex-scnu` 和 `scnuthesis-zsben2` 更适合作为场景补充与对照参考，而不是当前默认标准。

## 仍待确认的关键点

- 三套模板分别与当前学校实际规范的偏差有多大。
- 本科与研究生场景是否需要拆成两条独立入口。
- 非 LPPL 模板是否具备明确可再分发授权。
